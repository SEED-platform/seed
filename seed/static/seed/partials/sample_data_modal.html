Auto-Populate Data

Auto-Populate Data will insert sample records in your organization viewable from the Inventory Tab. The provided data demonstrates the following advanced capabilities of the SEED Platform:
  • Meter Data
    • PM Property IDs: 5766973; 5766975
    • Go To Detail Page []. Click the Meters tab at the top of the screen
  • Green Button Import
    • PM Property IDs: 5766975
    • Go To Detail Page []. Click the Meters tab at the top of the screen
  • Geocoding
    • PM Property IDs: 5766963; 5766969
    • Click the Map tab at the top of the screen
  • Merging [ ]
    • PM Property ID: 5766979
    • Go To Detail Page [] to view the history of the merged record
  • Property Footprint
    • PM Property ID: 5766965
    • Scroll-right to the field Property Footprint. Hover the mouse over the POLYGON coordinates to visualize the propertyâ€™s footprint
  • Tax Lots
    • PM Property ID: 5766961
    • Click the View by Tax Lot tab to see the paired tax lot record
  • Labels
    • PM Property ID: 5766967, 5766973, 5766977
    • Add Housing label in the Filter by label box at the top of the page to view records
  • BSyncr Analysis
    • PM Property ID: 5766973
    • Go To Detail Page []. Click the Actions drop down box at the top left of the screen. Click Run Analysis in the drop down menu. Name your analysis and enter Type as BSyncr. Create Analysis
Auto-Populate is only intended for organizations with no properties or tax lots. To continue you must first remove pre-existing data
